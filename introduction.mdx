# Welcome to Panopticon

Panopticon is an open-source framework for orchestrating multiple AI coding assistants. It provides a dashboard for monitoring agents, managing workspaces, and coordinating specialist agents for code review, testing, and merging.

## Features

- **Multi-Agent Dashboard** - Monitor all your AI agents in real-time
- **Workspace Management** - Create isolated workspaces for each issue
- **Specialist Agents** - Dedicated agents for review, testing, and merging
- **Skills System** - Reusable workflows and best practices
- **Issue Tracker Integration** - Works with Linear and GitHub Issues

## Quick Start

```bash
npm install -g panopticon-cli
pan setup
pan up
```

Visit [localhost:3010](http://localhost:3010) to see your dashboard.
