# Core Commands

Core system management commands for Panopticon.

## pan sync

Sync skills and configuration to AI tools.

## pan doctor

Check system health and configuration.

## pan up

Start the Panopticon dashboard and services.

## pan down

Stop the dashboard and services gracefully.

## pan restart

Restart the dashboard (frontend and API).

## pan health

Check the health of all services.
