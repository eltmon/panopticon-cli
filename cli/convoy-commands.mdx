# Convoy Commands

Orchestrate multiple agents working in parallel on related tasks.

## pan convoy start

Start a convoy of parallel agents.

## pan convoy status

Check status of all agents in a convoy.

## pan convoy synthesize

Synthesize results from convoy agents.
