# Agent Commands

Spawn, monitor, and control autonomous AI coding agents.

## pan work issue

Create workspace and spawn agent for an issue.

## pan work status

Show all running agents and their status.

## pan work tell

Send a message to a running agent.

## pan work kill

Stop a running agent.

## pan work approve

Approve agent work and merge changes.

## pan work done

Mark work as complete and update issue status.
